How to Make a Poster Room

STEp 1) Setup a space on GATHER.TOWN

Select Create New Space -> Start from blank and fill out the remaining information before pressing Open Mapmaker

STEp 2) Copy the space id

Once on the mapmaker page, the space ID is the URL past /mapmaker/. For the URL https://gather.town/mapmaker/0Y1vcO1iHRCS4Abw/demo, the Space Id is 0Y1vcO1iHRCS4Abw/demo

STEp 3) Create an API Key

Go to this link to generate an API key which is permenantly linked to the account. The key must belong to the same account of the space being edited

STEp 4) Fill out an event sheet

Fill out an excel spreadsheet that follows the template outline. The sheet has 3 sections:
Attendees - A list of names and emails that will be added to the guestlist if the space is private
tables/teams - A list of 'tables' which MUST include a presenter name and ID. All other columns are optional.
Rooms - A list of rooms that have up to 4 doors leading to other rooms, a size, and a list of tables (ID) to be added to the room.

STEp 5) Ready To GO! CLick the button below to get started
